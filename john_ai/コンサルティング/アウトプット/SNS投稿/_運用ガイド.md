# SNS投稿 運用ガイド

ナレッジを SNS の投稿ネタ・原稿に変換して蓄積・配信する場所。

## ファイルの作り方
- 投稿ネタ帳：`投稿カレンダー.md`（予定と実績を管理）
- 原稿：`YYYY-MM-DD_<媒体>_<テーマ>.md`（例：`2026-06-01_X_紹介の増やし方.md`）

## 原稿テンプレ（コピーして使う）
```
---
媒体: X / Instagram / LinkedIn / Threads
テーマ:
元ナレッジ: KH-001 など
状態: 下書き / 予約 / 投稿済
---
# フック（最初の一行で止める）
# 本文
# CTA（誘導：プロフ / 記事 / 無料相談 など）
ハッシュタグ:
反応メモ（投稿後）: いいね__ / 保存__ / 反応__
```

## コツ
- 1ナレッジ＝複数投稿に割る（連載化）。
- 失敗点・教訓（FL）は共感を呼びやすい。
- 反応が良かった表現はナレッジに還元する。
