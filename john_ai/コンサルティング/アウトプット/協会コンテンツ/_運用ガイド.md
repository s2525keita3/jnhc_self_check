# 協会コンテンツ 運用ガイド

マニュアル・フレームワークを、協会の教材・認定講座・会員向けコンテンツに体系化する場所。

## 置くもの
- カリキュラム全体像：`カリキュラム.md`
- 各講座教材：`講座/<番号>_<タイトル>.md`
- 認定基準・テスト：`認定/`

## 教材テンプレ
```
---
講座名:
対象レベル: 入門 / 実践 / 認定
元ナレッジ: MN-001, FW-001 など
到達目標:
---
# 学ぶこと
# 本編（手順・型）
# 演習 / チェックリスト
# 確認テスト
```

## ポイント
- マニュアル（MN）とフレームワーク（FW）が教材の中核素材。
- 体系（入門→実践→認定）で並べると講座商品として売りやすい。
- 起業塾のカリキュラムと素材を共有し、二重作成を避ける。
