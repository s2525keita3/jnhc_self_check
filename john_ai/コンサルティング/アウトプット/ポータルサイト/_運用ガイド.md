# ポータルサイト 運用ガイド

ナレッジをWeb記事・FAQ・匿名事例として蓄積し、サイトの元データにする場所。

## 置くもの
- 記事原稿：`articles/<slug>.md`
- FAQ：`faq.md`（QAナレッジをまとめて転記）
- 事例（匿名）：`cases/<slug>.md`

## 記事テンプレ
```
---
タイトル:
カテゴリ:
元ナレッジ: KH-001, FW-001 など
状態: 下書き / 公開
SEOキーワード:
---
# 導入（読者の悩み）
# 本文（ノウハウ）
# まとめ + CTA（無料相談 / 起業塾 / 協会）
```

## ポイント
- QAナレッジはそのままFAQに流用できる。
- 事例は必ず匿名化。数字はぼかすか一般化。
- 公開記事はSNS・起業塾・協会へ相互リンクして回遊を作る。
