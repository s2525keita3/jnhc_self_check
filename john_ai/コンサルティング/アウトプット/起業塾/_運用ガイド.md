# 起業塾 運用ガイド

ノウハウ・フレームワーク・失敗事例を、受講生向けのカリキュラム/レッスンに変換する場所。

## 置くもの
- カリキュラム：`カリキュラム.md`（全体の流れ）
- 各レッスン：`レッスン/<番号>_<タイトル>.md`
- 受講生向け課題：`課題/`

## レッスンテンプレ
```
---
レッスン番号:
タイトル:
元ナレッジ: KH-001, FW-001, FL-001 など
到達目標（このレッスンで出来るようになること）:
---
# 今日のテーマ
# 講義（ノウハウ + なぜ効くか）
# やってみよう（演習）
# よくあるつまずき（失敗点ナレッジから）
# 次回まで宿題
```

## ポイント
- 「ノウハウ＋失敗点」をセットにすると学習効果が高い。
- 実際のPDCA（匿名化）をケーススタディに使える。
- 協会コンテンツと素材を共有する。
