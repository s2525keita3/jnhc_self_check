# 商品化 運用ガイド

ナレッジ資産を有料商品（教材・テンプレ・講座・コミュニティ）に企画・設計する場所。

## 置くもの
- 商品企画：`企画/<商品名>.md`
- LP草案：`LP/<商品名>.md`
- 価格設計：`価格設計.md`

## 商品企画テンプレ
```
---
商品名:
形態: 教材 / テンプレDL / 講座 / コミュニティ / 顧問
元ナレッジ: KH-xxx, FW-xxx ...
ターゲット:
提供価値（Before→After）:
価格:
状態: 構想 / 制作中 / 販売中
---
# 解決する悩み
# 中身（提供物）
# 差別化（なぜこれが選ばれるか）
# 価格と根拠
# 販売導線（SNS→LP→決済）
```

## ポイント
- フレームワーク（FW）はワークシート/テンプレ商品にしやすい（即金性が高い）。
- 商品は「無料（SNS）→ 安価（テンプレ）→ 中価格（講座/起業塾）→ 高価格（顧問/協会認定）」の階段で設計する。
- 売れた／売れなかった理由はナレッジ（成功＝KH、失敗＝FL）に還元する。
