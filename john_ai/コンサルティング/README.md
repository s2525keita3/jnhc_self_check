# コンサルティング ナレッジシステム

クライアント支援の「記憶・整理・分析・再利用」を回す本体です。
**まず最初に [`_システム/運用マニュアル.md`](_システム/運用マニュアル.md) を読んでください。**

---

## 3つのレイヤー

| レイヤー | フォルダ | 役割 | 粒度 |
|---|---|---|---|
| **記憶** | `クライアント/` | 個社で何が起きたかを正確に残す | 個別・時系列 |
| **知の資産** | `ナレッジベース/` | 個社を越えて使える知見に昇華 | 抽象・横断 |
| **収益化** | `アウトプット/` | 資産をコンテンツ・商品に転換 | 公開・商用 |

情報は基本的に **記憶 → 知の資産 → 収益化** の方向で上がっていきます。
PDCAの学びは逆向きに各レイヤーへ反映されます。

## クイックスタート

- **新規クライアントを追加** → `_システム/運用マニュアル.md`「クライアント追加」を参照
- **会議の議事録を残す** → `テンプレート/議事録.md` をコピーして `クライアント/<会社名>/01_議事録/` へ
- **知見を貯める** → `_システム/ナレッジ抽出ルール.md` に従い `ナレッジベース/` へ
- **コンテンツ化する** → `アウトプット/README.md` を参照
