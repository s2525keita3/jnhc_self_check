# 出演者（じょん）写真素材

監修・講師 **渋谷 慶太（じょん）** ／ JNHC-MRA 代表理事。
サムネイル・SNS・営業資料で「顔」として使う本人写真。

> ⚠️ チャットに貼られた写真はファイル保存できなかったため、下記の名前で
> **この `assets/people/jon/` にドロップ**してください。

## 大原則（必読）

- **本人の実写を使う。顔を gpt-image で生成・合成・改変しない。**
  実在の本人なので、AIで“それっぽい顔”を作るのは似顔の捏造になりNG。
  gpt-image は**背景**を作る係。顔は常に実写のまま。
- 推奨ワークフロー = **切り抜き合成**:
  1. 本人写真の背景を除去 → 透過PNGの切り抜きを作る
  2. `prompts/youtube.md` のブランド背景を gpt-image で生成
  3. `scripts/compose.py` で「背景＋切り抜き＋ロゴ」を合成
  4. 文字（特大コピー）は編集ソフトで後乗せ
- 背景だけ差し替えたい場合のみ `image_gen.py edit` を使うが、**顔が変わって
  いないか必ず目視確認**。少しでも変われば切り抜き合成に戻す。

## 命名規則

| ファイル名 | 内容 | 向く用途 |
| ---------- | ---- | -------- |
| `jon-suit-desk.jpg` | 紺スーツ・デスク＋PC・横位置・左を見て笑顔 | 横長サムネ/バナー。右の余白に文字 |
| `jon-suit-front.jpg` | 紺スーツ・縦・正面 | 縦サムネ/SNS、切り抜き元 |
| `jon-suit-typing.jpg` | 紺スーツ・縦・タイピング | 切り抜き元、変化 |
| `jon-scrubs-a.jpg` | 黄緑スクラブ・縦・正面 | 現場/看護トピック、親しみ |
| `jon-scrubs-b.jpg` | 黄緑スクラブ・縦・正面（別カット） | 同上、差し替え |
| `jon-suit-cutout.png` | スーツ姿の**透過切り抜き** | 合成用（最重要） |
| `jon-scrubs-cutout.png` | スクラブ姿の**透過切り抜き** | 合成用 |

## 2つの“顔”の使い分け

- **スーツ（紺）** = 経営者・講師・権威。採用設計／経営／セールス／講座系の
  サムネ・資料に。信頼と説得力。
- **スクラブ（黄緑）** = 現場の看護師・親しみ。ケア／スタッフ向け／現場あるある
  系の動画・SNS に。柔らかさと共感。

> ⚠️ スクラブの黄緑（ライムグリーン）はブランド色ではない。ブランド背景
> （ネイビー/ブルー/ティール）と合わせると色がぶつかるので、スクラブ写真を使う
> サムネは背景を**白〜淡色 or ティールグリーン低彩度**に寄せて馴染ませる。

## 切り抜きPNGの作り方（任意）

```bash
# rembg を使う例（要: pip install rembg onnxruntime）
rembg i assets/people/jon/jon-suit-front.jpg assets/people/jon/jon-suit-cutout.png
```
Photoshop / Canva / Keynote の「背景を削除」でも可。エッジを綺麗に。
