# SNS プロンプト集（背景・スポットイラスト／文字なし）

Instagram カルーセル / X 図解用の背景・装飾・スポットイラスト。文字は後乗せ。
比率は用途で選ぶ（4:5=`portrait`→トリミング, 1:1=`square`, 16:9=`landscape`）。
〔BRAND〕は `prompts/sales.md` 冒頭のブランド節と同じ。

---

## 1. カルーセル表紙の背景（4:5・余白多め）
```
A clean Instagram carousel cover background (4:5 vertical), mostly white, soft
pale-blue light gradient from the top, a delicate translucent watercolor
botanical sprig (eucalyptus/olive, sage #9DBFA8 + powder blue #CFE6E8, low
opacity) in the bottom-right. Wide empty center for a strong headline later.
Calm, trustworthy. No text, no logo, no people. 〔BRAND〕
```

## 2. カルーセル中ページの背景（統一感・控えめ）
```
A minimal Instagram carousel inner-page background (4:5 vertical), almost all
white with a thin blue accent line at the top and a very faint botanical hint
bottom-right, maximum empty space for a numbered step diagram and text. Quiet
and consistent across slides. No text, no logo. 〔BRAND〕
```

## 3. 1枚図解の背景（X・16:9）
```
A clean 16:9 background for a single explanatory diagram post: near-white with a
subtle blue grid feeling, a soft botanical sprig bottom-right, generous empty
space for a comparison table or before/after diagram to be added later. Tidy,
credible. No text, no logo. 〔BRAND〕
```

## 4. キーメッセージ / 名言（1:1・余白主役）
```
A calm 1:1 quote-style background, mostly white with a generous central empty
space, a soft blue light gradient in one corner and a delicate low-opacity

text, no logo. 〔BRAND〕
```
